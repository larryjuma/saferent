SafeRent is a full-stack **property rental platform** built with the **MERN stack** (MongoDB, Express, React, Node.js).  
It allows users to browse rental properties, view detailed property information with a modern carousel UI, and submit rental requests.

This project showcases **full-stack development skills, responsive design, and real-world web app functionality**‚Äîperfect for your portfolio or resume.

## üåü Features

### User Features

- Browse all available rental properties on a responsive dashboard.
- View property details with multiple images in a carousel.
- See property information: price, bedrooms, bathrooms, type, and description.
- Submit rental requests (requires login).
- Mobile-friendly and responsive design.

### Landlord Features

- Create, update, and delete properties (admin functionality available via backend API).
- Manage property listings.

### UI & UX Highlights

- Modern, clean design with **glassmorphism cards**.
- Smooth **carousel animation** for property images.
- Hover effects and interactive buttons.
- Fully responsive across mobile, tablet, and desktop.
